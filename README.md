# Horror Riding Hood

![Horror Riding Hood](horror_riding_hood.png)

## 📝 Description

**Horror Riding Hood** est un jeu d'horreur en 3D réalisé dans le cadre d'un **projet universitaire**.

Inspiré du célèbre conte, vous incarnez le Petit Chaperon Rouge parti chercher des ingrédients pour sa grand-mère. Mais attention... les loups rôdent dans les parages et vous devez à tout prix leur échapper !
